Spreadsheet Operations

Overview

jXLS currently does not provide many excel sheet manipulations but you may want to look at POI Objects Access section to see how you can use core POI objects like Sheet and Workbook to perform some actions on sheet and workbook.

Spreadsheet removing

You can remove some sheet dynamically in your Excel export with the method void setSpreadsheetsToRemove(String[] names) of XLSTransformer class. It takes the names of spreadsheets to remove.

Spreadsheet renaming

Sometimes it is required to set sheet name dynamically. It can be done with void setSpreadsheetToRename(String name, String newName) method of XLSTransformer. This method takes template spreadsheet name and schedules it to be renamed into newName. The method may be invoked multiple times.
